# 📝 Changelog — SiteForge

Все заметные изменения в проекте SiteForge.

---

## [1.0.0] — Базовый конструктор

### Добавлено
- Визуальный конструктор сайтов в одном HTML-файле
- Левая панель с настройками, разделённая по секциям
- Предпросмотр сайта справа в реальном времени
- Базовые настройки:
  - Фон сайта (цвет, градиент, изображение)
  - Шапка (логотип, меню)
  - Hero-блок (заголовок, подзаголовок, кнопка)
  - Карточки услуг (3 карточки с названием и описанием)
  - Подвал (копирайт, соцсети)
  - Шрифты (заголовки, текст)
- Кнопка «Скачать сайт» — сохраняет готовый HTML-файл
- Переключение режимов просмотра: десктоп, планшет, телефон

---

## [2.0.0] — Реворк дизайна и UX

### Добавлено
- Полностью новый интерфейс в тёмных тонах 
- Иконки-эмодзи в разделах меню для наглядности
- Бейджи с подсказками в пунктах меню
- Анимации при переключении разделов
- Интерактивный гайд для новых пользователей (показывается один раз)
- Автосохранение всех настроек в localStorage браузера
- Всплывающие уведомления (toast) при действиях

### Изменено
- Разделы настроек теперь в виде компактного вертикального меню
- Кнопки унифицированы в едином стиле

### Авторское право
- В интерфейс добавлено: **© CayPlay 2026. Все права защищены.**
- В подвал каждого скачанного сайта: **Создано в SiteForge © CayPlay 2026**

## [3.0.0] — Интерактивный шаблон

### Добавлено
- Меню в шаблоне стало полностью рабочим:
  - Пункты меню автоматически привязываются к секциям через якоря (`#id`)
  - Плавная прокрутка при клике (`scroll-behavior: smooth`)
  - Транслитерация русских названий в ID секций
- Кнопка в Hero ведёт на секцию с карточками услуг
- Автоматическая генерация дополнительных секций для пунктов меню
- Для «Портфолио» и «Контактов» — отдельные настройки контента

### Исправлено
- Убран эффект «конструктор в конструкторе» — шаблон внутри iframe теперь автономный

---

## [4.0.0] — Полная кастомизация

### Добавлено
- Настройки **всех кнопок** на сайте:
  - **Кнопка Hero**: текст, ссылка (якорь), цвет фона, цвет текста, скругление, размер шрифта
  - **Кнопки в карточках**: возможность включить/выключить, текст, ссылка, цвета
  - **Дополнительная кнопка 1**: после Hero-блока (по умолчанию «Связаться с нами»)
  - **Дополнительная кнопка 2**: перед подвалом (по умолчанию выключена)
- Полная кастомизация **соцсетей в подвале**:
  - Формат: `Название:ссылка:иконка`
  - 3 стиля отображения: текст с иконками, кнопки, только иконки
  - 3 размера: маленький, обычный, крупный
- Настройки иконок карточек (эмодзи)
- Настройка тени карточек (мягкая, средняя, сильная, без тени)
- Закрепление шапки при скролле (вкл/выкл)
- Цвет ссылок меню при наведении
- Подчёркивание ссылок меню при наведении (вкл/выкл)
- Фоновое изображение для Hero-блока
- Высота Hero-блока (ползунок 300-900px)
- Количество карточек в ряду (2, 3 или 4)
- Общий акцентный цвет сайта

### Изменено
- Расширена левая панель (с 6 до 9 разделов настроек)
- Настройки сгруппированы логически: кнопки отдельно, соцсети отдельно

## [5.0.0] — Исправление багов 

### Исправлено
- **Убраны черточки-разделители** `·` между соцсетями в подвале
  - Теперь соцсети отображаются через flexbox с `gap: 14px` — чистые отступы без лишних символов
- **Автоматическое добавление `https://`** к ссылкам соцсетей, если протокол не указан
  - Больше никаких битых ссылок вида `//vk.com` — теперь всегда `https://vk.com`
- Стиль соцсетей по умолчанию изменён с «текст» на «кнопки» (выглядит современнее)
- В iframe добавлен атрибут `sandbox="allow-same-origin allow-scripts allow-top-navigation"` для корректной работы внешних ссылок

### Изменено
- Подсказка в поле соцсетей обновлена: «Можно писать ссылки с http:// или без»
