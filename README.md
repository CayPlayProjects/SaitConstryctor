# ✨ SiteForge — Визуальный конструктор сайтов

**SiteForge** — это конструктор сайтов, который работает прямо в браузере. Тебе не нужно знать HTML, CSS или какой-либо язык программирования. Просто открываешь файл, настраиваешь всё через удобную панель, жмёшь «Скачать» — и готовый сайт у тебя!

---

## 🚀 Что внутри

- Полноценный одностраничный сайт (лендинг) с шапкой, Hero-блоком, карточками услуг, дополнительными секциями и подвалом
- Удобная панель управления слева — все настройки разбиты по разделам
- Живой предпросмотр — все изменения видны сразу справа
- Можно переключать режим просмотра: 💻 Десктоп / 📱 Планшет / 📱 Телефон
- Автосохранение всех настроек в браузере — ничего не пропадёт

---

## 🎨 Что можно настроить

### Фон и общие стили
- Тип фона: сплошной цвет, градиент или своё изображение
- Основной цвет текста, акцентный цвет ссылок и кнопок

### Шапка и меню
- Логотип (текст), цвет фона и текста
- Пункты меню (через запятую) — автоматически создаются секции и якоря
- Закрепление шапки при скролле, эффекты при наведении

### Главный блок (Hero)
- Заголовок, подзаголовок, фоновое изображение, высота блока

### Кнопки
- **Главная кнопка** — текст, ссылка (якорь #), цвет фона и текста, скругление, размер шрифта
- **Кнопки в карточках** — можно включить/выключить, настроить текст, ссылку, цвета
- **Дополнительные кнопки** — до двух штук в любом месте страницы

### Карточки услуг
- Название, описание, иконка (эмодзи) для каждой из трёх карточек
- Цвет фона, количество в ряду, тень

### Дополнительные секции
- Контент для «Портфолио», «Контактов» и любых других пунктов меню

### Соцсети в подвале
- Формат: `Название:ссылка:иконка` (можно без https:// — добавится автоматически)
- 3 стиля: текст, кнопки или только иконки
- 3 размера

### Типографика
- Шрифты для заголовков и текста
- Скругление элементов, масштаб заголовков, тень карточек

## 📥 Как пользоваться

1. **Открой файл `constructor.html`** в любом браузере (Chrome, Firefox, Edge)
2. **Слева выбери раздел** — нажми на «Фон», «Шапка», «Кнопки» и т.д.
3. **Настрой всё под себя** — меняй текст, цвета, ссылки
4. **Нажми «✨ Применить»** — справа обновится предпросмотр
5. **Нажми «💾 Скачать HTML»** — получи готовый сайт одним файлом

Всё! Скачанный файл можно открыть в браузере, загрузить на хостинг или отправить кому угодно.
